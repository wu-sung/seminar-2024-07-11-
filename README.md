## GPU에 맞는 NVIDIA 드라이버 설치
Q. NVIDIA 드라이버 왜 필요할까요? <br>

A. 많은 연산을 cpu에서만 담당하게 되면 cpu가 해야할 컴퓨터의 프로세스 관리 업무를 효율적으로 수행하지 못할 뿐더러 요구되는 연산 처리에 대한 소요시간이 증가하기 때문에 gpu를 활용해야 하고 이 때 필요한게 gpu 드라이버
